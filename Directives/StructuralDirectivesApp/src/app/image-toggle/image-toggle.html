How NgIf Works

What is *ngIf?

*ngIf is a structural directive that conditionally adds or removes elements from the DOM. When the condition is false, Angular completely removes the element and its children from the DOM (not just hiding it with CSS).

In this example:

  • showImage is a boolean variable
  • *ngIf="showImage" shows the image when true
  • *ngIf="!showImage" shows a message when false
  • The button calls toggleImage() to flip the boolean

Key Points:

  • Elements are removed from the DOM, not just hidden with CSS
  • This frees up memory and resources
  • The asterisk (*) is shorthand for <ng-template>
  • Great for conditional rendering based on user state, permissions, or data availability
