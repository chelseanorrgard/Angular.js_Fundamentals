How Text Transform Works

The Directive Features:

  • uppercase: Converts all text to UPPERCASE
  • lowercase: Converts all text to lowercase
  • capitalize: Capitalizes The First Letter Of Each Word
  • reverse: sdrawkcab txet eht sesreveR
  • alternate: AlTeRnAtEs BeTwEeN uPpEr AnD lOwEr CaSe

How It Works:

  • @Input(): Accepts the transformation type from the template
  • ElementRef: Gets the DOM element's text content
  • Renderer2: Safely updates the element's text content
  • transformText(): A private method that performs the actual transformation

Usage Examples:

<h1 appTextTransform="uppercase">hello world</h1>
<p appTextTransform="reverse">This will be reversed</p>
<span appTextTransform="capitalize">hello world</span>

Bonus: You can combine multiple directives! Use both appTextColor and appTextTransform on the same element for colorful transformed text!

How Text Color Works

The Directive:

  • @Input(): Accepts a color value from the template
  • ElementRef: References the DOM element where the directive is applied
  • Renderer2: Safely manipulates the DOM to set the color style
  • ngOnInit(): Applies the color when the component initializes

Usage:

<h1 appTextColor="green">This is green!</h1>

The directive accepts any valid CSS color value: named colors (red, blue), hex codes (#ff0000), RGB values (rgb(255, 0, 0)), or any other CSS color format!
