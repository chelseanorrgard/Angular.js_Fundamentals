Angular Built-in Pipes Demo

This page demonstrates various built-in Angular pipes for formatting data.

DatePipe

Original: {{ today }}

Short date: {{ today | date:'short' }}

Full date: {{ today | date:'fullDate' }}

Custom format: {{ today | date:'MMMM d, y, h:mm a' }}

CurrencyPipe

Original: {{ amount }}

USD currency: {{ amount | currency:'USD':'symbol':'1.2-2' }}

EUR currency: {{ amount | currency:'EUR':'symbol':'1.2-2' }}

DecimalPipe

Original: {{ amount }}

Formatted (1.2-2): {{ amount | number:'1.2-2' }}

Formatted (1.0-0): {{ amount | number:'1.0-0' }}

PercentPipe

Original: {{ percent }}

As percentage: {{ percent | percent }}

With 2 decimals: {{ percent | percent:'1.2-2' }}

UpperCase/LowerCase Pipes

Original: {{ message }}

Lowercase: {{ message | lowercase }}

Uppercase: {{ message | uppercase }}

JsonPipe

User object as JSON:

{{ user | json }}
