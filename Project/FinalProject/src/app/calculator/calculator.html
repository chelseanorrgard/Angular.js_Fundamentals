How It Works

This calculator performs basic arithmetic operations: addition, subtraction, multiplication, and division. It's built using Angular event binding to handle button clicks and TypeScript methods to perform calculations.

Features include decimal support, error handling for division by zero (because apparently that breaks math), and a clear button for when you inevitably mess up.

Fun fact: I could have just used eval(), but then my code would be as cursed as dividing by zero.
