Rick & Morty Character Explorer

Because if I'm using APIs, it might as well involve interdimensional chaos

Why Rick and Morty?

When you're learning how to integrate APIs into your Angular application, you have a choice. You can use some boring weather API and pretend you care about the temperature in Vaasa, or you can dive into the multiverse with Rick and Morty. I chose chaos.

This component uses Angular's HttpClient to fetch data from the Rick and Morty API, handles asynchronous operations with Observables, and displays random characters because predictability is for people without portal guns.
